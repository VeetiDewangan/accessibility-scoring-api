# Dynamic Accessibility Scoring Engine ♿

An AI-powered API developed for accessibility assessment, originally created for the **Aptiv Hackathon**. This engine combines physical infrastructure metrics with NLP-based sentiment analysis to provide a holistic accessibility score for venues.

## 🚀 Features
- **Machine Learning Core**: Uses a Linear Regression model to predict accessibility scores (0-100).
- **NLP Integration**: Leverages `NLTK VADER` to analyze the sentiment of user reviews.
- **FastAPI Backend**: High-performance asynchronous API for real-time predictions.
- **Web Dashboard**: A clean HTML5/Bootstrap interface for user interaction.

## 🛠️ Tech Stack
- **Languages**: Python, HTML/JS
- **Frameworks**: FastAPI, Bootstrap 5
- **AI/ML**: Scikit-learn, NLTK, Joblib
- **Environment**: Python 3.12+
